word-o-mat v1.0

word-o-mat is an extension for RoboFont that generates words for use in type sketching, spacing, testing etc. This is my first RF extension, supplied without any promises and all that jazz. If you find any bugs or have suggestions for future development, please either fix them or send me a line, thx :)

Interface Elements

Word Count / Length

The first row lets you specify the amount of words that should be generated. The boxes in the second row let you adjust minimal and maximal length of created words. Both these numbers are inclusive – so specifying 3 and 5 for instance will yield three, four and five letter words.

Required Characters

Enter characters that must be contained in all returned words (a–z, A–Z; comma- or space-delimited list).

Groups

You can define up to three groups or sets of characters, and word-o-mat will look for words that contain at least one character from each of the specified groups. You can choose from presets (letters with ascenders, descenders, arches, diagonals, and ball-and-stick letters) or just type in your own (a–z, A–Z; comma- or space-delimited list).

Excluded Characters

Enter characters that must not be contained in the returned words (a–z, A–Z; comma- or space-delimited list).

Limit to characters available in current font

By default, word-o-mat does not limit the characters occurring in the words it finds. If you want to use word-o-mat for testing your existing font, you can tick this box to make word-o-mat exclude any words using letters you don’t have (yet). (Obviously, this option is only available if you have a font open.)

No repeating characters

Excludes words in which one or more characters occur more than once. Might be useful when generating words for sketching, etc.

Randomize output

Gives a random list of words every time the script is run. If this is unchecked, word-o-mat will simply return the first words it finds in the word list (see below).

MEQ (Most Expected Questions)

Where is the output?

If you have a Space Center window open in RoboFont, the word-o-mat output goes directly there, using the current font. If no Space Center (or no font) is open, word-o-mat will output to the Output Window.

Where do the words come from?

word-o-mat uses a comprehensive English-language word list originally made for crossword solvers, the UK Advanced Cryptics Dictionary (UKACD). At around 250.000 lines, it is a fairly long freely redistributable word list. Note that the UKACD also includes expressions that span multiple words (with a space); word-o-mat disregards those entries, since it looks strictly for words.

Can I use a different word list?

You can. It needs to be a text file with words on separate lines; drop it inside the /resources/ folder in your word-o-mat installation, and set the fileName and contentLimit variables in the main word-o-mat.py script accordingly.

Can I use word-o-mat for other languages?

There are two things to consider. One is the word list that word-o-mat uses to find words. This can be replaced, see above, so you are free to use a list using words in a language other than English. The other consideration however is that word-o-mat does not yet support specifying characters outside the basic a–z/A–Z range as search criteria. This may be fixed in future versions if there is interest.

word-o-mat makes my RoboFont reeeeally slow.

Are you outputting hundreds of words into your Space Center? Maybe try a few less.

It says I have no font open, but I do!

Did you open a font after launching word-o-mat? It currently only checks for open/current fonts once when it’s launched, so you’ll have to close its little window and open it again. Sorry!
